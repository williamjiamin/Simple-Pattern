# Simple Pattern

**Custom Vibrator Patterns & Private Vault**

Simple Pattern is an iOS app that lets you create, customize, and enjoy haptic vibration patterns for relaxation and entertainment. Featuring motion-reactive and audio-reactive modes, a private media vault, and a pattern studio for designing your own haptic experiences.

## Features

- Manual vibration control with customizable preset patterns
- Motion-reactive mode — control haptics by tilting your device
- Audio-reactive mode — sound-responsive vibration patterns
- Tap to Train — create patterns by tapping rhythms
- Pattern Studio — design custom haptic patterns
- Private Media Vault — securely store photos and videos on-device
- Gallery slideshow with auto-play, loop, and shuffle
- Adaptive recommendations based on your preferences
